Convert-DXF-to-PIXI-using-the-Vue-framework

my versions:

    npm  10.8.2
    node 18.20.7

operating steps:

    npm i

    npm run dev

example:

    /public/temp.dxf

limits:

The dxf file should be in the 2018 version, and its length should ideally not exceed 5000 pixels

Note:

The dxf folder under the utils directory of this project is a secondary encapsulation of the dxf third-party library
